# Cosmic - PetUniverse

Project developed for the Fundamentals of Web Systems course.

## Description

Cosmic - PetUniverse is an online pet shop platform created to present products and services for pets.

## Features

- Navigation by sections on the same page
- Product visualization by category
- Display of product images, descriptions, and prices
- Display of pet shop services
- Information about telephone ordering for services
- Contact area with pet shop data

## Page Structure

- Header with navigation
- Presentation section
- Accessories section
- Non-perishable pet food section
- Hygiene and cleaning section
- Services section
- Contact section
- Footer with authorship, contact information, and navigation

## Technologies Used

- HTML

## Publication

The project must be published on GitHub and hosted on GitHub Pages.

# Cósmica - PetUniverse

Projeto desenvolvido para a disciplina de Fundamentos de Sistemas Web.

## Descrição

A Cósmica - PetUniverse é uma plataforma online de petshop criada para
apresentar produtos e serviços para animais de estimação.

## Funcionalidades

-   Navegação por seções na mesma página
-   Visualização de produtos por categoria
-   Exibição de imagem, descrição e valor dos produtos
-   Exibição dos serviços do petshop
-   Informação sobre tele-busca nos serviços
-   Área de contato com dados do petshop

## Estrutura da página

-   Cabeçalho com navegação
-   Seção de apresentação
-   Seção de acessórios
-   Seção de rações não perecíveis
-   Seção de higiene e limpeza
-   Seção de serviços
-   Seção de contato
-   Rodapé com autoria, contato e navegação

## Tecnologias utilizadas

-   HTML

## Publicação

O projeto deve ser publicado no GitHub e hospedado no GitHub Pages.
