# Cósmica - PetUniverse - Versão final ajustada

Projeto desenvolvido para a disciplina de Fundamentos de Sistemas Web.

## Tecnologias utilizadas
- HTML
- CSS
- Bootstrap
- JavaScript

## Funcionalidades implementadas
- Página inicial com produtos e serviços
- Página inicial sem formulários, focada em apresentação e navegação
- Botões de agendamento nos cards de serviços
- Página separada para cadastro e agendamento
- Cadastro do cliente
- Cadastro do pet
- Escolha do serviço, data e horário do agendamento
- Validação de CPF
- Máscara de CPF e telefone
- Verificação de horários passados
- Simulação de horários ocupados
- Feedback visual de erro e sucesso
- Acessibilidade com alt, labels e aria-live

- index.html

- agendamento.js
